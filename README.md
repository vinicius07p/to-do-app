#  To-Do List com HTML e CSS

##  Descrição

Este projeto consiste em uma aplicação simples de lista de tarefas (To-Do List), desenvolvida utilizando HTML e CSS, com foco na prática de Engenharia de Prompt aplicada ao desenvolvimento de software.

A aplicação permite visualizar tarefas com diferentes níveis de prioridade (alta, média e baixa) e simula interações como conclusão de tarefas.

---

##  Objetivo

Aplicar conceitos de:

* Estruturação de páginas com HTML
* Estilização com CSS
* Engenharia de Prompt
* Pensamento crítico no uso de IA

---

##  Tecnologias Utilizadas

* HTML
* CSS
* JS

---

##  Funcionalidades

* Campo para inserção de tarefas
* Seletor de prioridade (Alta, Média, Baixa)
* Lista de tarefas
* Diferenciação visual por prioridade
* Simulação de conclusão de tarefas

---

##  Uso de Inteligência Artificial

A IA foi utilizada como ferramenta de apoio no desenvolvimento, atuando como um assistente para:

* Planejamento da estrutura da aplicação
* Sugestões de código HTML e CSS e JS
* Revisão de boas práticas

Todos os códigos gerados foram analisados, compreendidos e adaptados conforme necessário.

🧠 Reflexão Crítica

Durante o desenvolvimento deste projeto, foi possível compreender melhor como a Inteligência Artificial pode ser utilizada como uma ferramenta de apoio no processo de criação de software.

Um dos principais aprendizados foi a importância da Engenharia de Prompt. Ao estruturar melhor os comandos com persona, contexto, tarefa e formato, as respostas da IA se tornaram mais úteis e alinhadas com o objetivo do projeto.

Outro ponto relevante foi perceber que a IA não substitui o conhecimento do desenvolvedor. Em diversos momentos, foi necessário revisar, adaptar e até corrigir sugestões geradas, reforçando a necessidade de entendimento do código antes de utilizá-lo.

Também foi possível desenvolver um olhar mais crítico sobre organização, semântica HTML e boas práticas de CSS, além de pensar na escalabilidade do projeto para futuras implementações com JavaScript.

Por fim, a atividade mostrou que o uso consciente da IA pode acelerar o aprendizado, desde que o estudante mantenha o protagonismo no processo e busque compreender cada decisão tomada.
